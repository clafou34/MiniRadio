# Using Miniradio

## Installation
Create your MiniRadio folder
`mkdir MiniRadio`

Cop

Create virtual environnement
`python3 -m venv </your/path/env>`
`source </your/path/env>/bin/activate`

Install dependences
`pip install bottle`
`pip install python-mpd2`

## Execution
`source </your/path/env>/bin/activate`
`pyhton3 /MiniRadio/path/application.py`
